PyPI publishing is handled by GitHub Actions trusted publishing.

PyPI trusted publisher settings for this project:
	project: rf-info
	owner: cosmicc
	repository: rf_info
	workflow: publish.yml
	environment: leave blank

Release steps:
	git add -A
	git commit -m <version>
	bump2version minor (or patch, major)
	git push
	git push --tags

Pushing a v*.*.* tag triggers .github/workflows/publish.yml, which runs tests,
builds the package, checks the distributions with twine, and uploads to PyPI.
